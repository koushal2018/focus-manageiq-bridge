Connect a data source.

Onboarding a cloud account is a registry entry plus a credential reference — never a code change. Register a source, the dispatcher normalizes it to FOCUS and loads it.

PoC scope
In this PoC a registered source reads a local synthetic export, so registering one and re-running the dispatcher actually produces rows. It does not yet perform a live cloud fetch with the credential you supply — that adapter half (discover() over S3/blob + Secrets Manager) is wired in production. The credential field stores a reference string only.
