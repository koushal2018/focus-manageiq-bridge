Source & scope
Data joined from FOCUS v1.3 billing exports (AWS / Azure / OCI) and the on-prem ManageIQ inventory + utilization warehouse. Cloud cost is native FOCUS; on-prem cost is a per-resource recharge model and is labelled accordingly. All values are normalised to USD (AED pegged at 3.6725). All data is synthetic; identifiers prefixed DEMO- are intentionally non-production. Verdicts below describe what FOCUS can answer — not what we wish it could.
